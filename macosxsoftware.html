Stock Predictor

This is a GUI tool built with Swift and Python that lets the users look up any stock or index symbol, and the computer will automatically use the real-time market data to generate a graph of how the stock will do for the rest of the year. The Swift is for the UI and the compiling, and the python is for all the data scraping, server side requests, and organizing the JSON data.

Image Uploader

This is the cheapest Mac App that lets you get direct online links to your local images. Through many services, you can easily get an online link for any image you have and share it with your friends. Also, the app saves all your history with the images and links which you can choose to delete anytime.
